Featured Projects

1. Fixed Income

Objective: benchmark linear-Gaussian filtering behavior for yield-state proxy estimation under observation noise.

  • Uses NumPy-generated latent state and noisy observations (100 time steps).
  • Notebook framing references Riccati vs Joseph covariance-update baseline.
  • Produces trajectory visualization using Matplotlib for quick residual inspection.

Concrete result: observation-to-state RMSE = 0.761 over the 100-step run.

2. Market Risk

Objective: compare nonlinear filter consistency behavior with NEES-based calibration diagnostics.

  • Defines 60-step EKF/UKF NEES trajectories with an explicit 95% consistency band.
  • Uses an analytical stress-shape setup to expose overconfidence decay patterns.
  • Plots EKF and UKF traces against the same acceptance band for direct comparison.

Concrete result: UKF remains in-band for 28/60 steps, EKF for 19/60; initial NEES values are 3.532 (UKF) and 4.682 (EKF).

3. Sentiment

Objective: evaluate high-dimensional inference trade-offs for a signal-assimilation proxy pipeline.

  • Compares Bootstrap PF, LEDH, and Kernel-PFF RMSE across dimensions 16 to 224.
  • Encodes model-performance trends as explicit arrays for reproducible benchmarking.
  • Provides a fast sanity-check scaffold before integrating external NLP feeds.

Concrete result: at dimension 224, Kernel-PFF RMSE is 2.25 vs PF at 3.20 (29.7% lower).
